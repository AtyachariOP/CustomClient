# Architecture Decision Records

All Architecture Decision Records (ADRs) for Custom Client v1.0 have been centralized and codified within the `docs/architecture/blueprint-v2.0.md` document.

Please refer to that document for the canonical history and rationale behind the 12 core architecture decisions (IPC, File System, Process Model, Zustand vs Redux, Error Handling, etc).

Future ADRs (v1.1+) should be placed in this directory following standard ADR numbering (e.g. `ADR-013-cloud-sync.md`).
