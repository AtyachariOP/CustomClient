# Custom Client

Custom Client is an upcoming 100% configurable open-source Minecraft client.

## Features
- **In-Game Editor:** Modders can write scripts directly in the client to add new features without recompiling Java.
- **Customizable UI:** Edit the Homescreen, Esc screen, and Options menus completely with built-in layout parsers.
- **Marketplace:** Built-in marketplace to upload, buy, and sell themes and features.
- **Default Mods:** Includes basic features like FPS display, CPS display, and Keystrokes.

## File Structure
- `/client`: The core Minecraft Client mod (Fabric/MCP) written in Java.
- `/backend`: The Marketplace backend written in Node.js.
- `/docs`: Documentation for players and modders.
